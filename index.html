HTML & CSS Cheatsheet

Common HTML Tags

Tag Name Description
<h1-h6> Heading Used to capture attention like the heading of a newspaper
<p> Paragraph Contains large blocks of text
<div> Division Used to divide a page into sections
<span> Span Used to contain short text
<em> Emphasize Changes the font style to italic
<strong> Strong Changes the font weight to bold
<a> Link Creates links to other sites or elements on the same page
<img> Image Allows you post images on a site
<video> Video Allows you to post videos on a site

HTML Tables Tags

Tag Name Description
<table> Table Element The element the table is wrapped in
<thead> Table Head The element that the header of the table is wrapped in
<th> Table Header The heading rows of the table
<tr> Table Row The container for the table rows
<td> Table Data The container for the table rows
<tfoot> Table Footer The element that the footer of the table is wrapped in

Table Attributes

Attribute Description
colspan="#" Allows a data point to span across multiple columns
rowspan="#" Allows a data point to span across multiple rows

Common CCS Styles

Style Description
font-family Changes the font of the text
font-weight Changes how bold the text appears
color Changes the color of the foreground
background-color Changes the color of the background
